## Run Locally


To Clone the repository to your local machine:

```bash
git clone  https://github.com/anshumancodes/Javascript-learnings.git 
  
```
To run the file locally:

```bash
node  <filename>
 ```

install prompt-sync by running:

```bash
 npm install prompt-sync 
 ```
